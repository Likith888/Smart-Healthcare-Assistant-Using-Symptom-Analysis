# Smart Healthcare Assistant Using Symptom Analysis

## Overview
The Smart Healthcare Assistant is an AI-based system designed to help users get basic healthcare guidance using technology. The system analyzes user inputs such as symptoms and provides useful suggestions, basic health information, and guidance for possible medical conditions.

This project demonstrates the application of Artificial Intelligence and Machine Learning in the healthcare domain to assist users in understanding their health conditions quickly and efficiently.

## objectives
 - Provide quick health-related guidance based on symptoms.
 - Demonstrate the use of AI in healthcare applications.
 - Build a simple and user-friendly healthcare assistant system.

## Features
- Symptom based disease prediction
- Preventive healthcare suggestions
- AI-driven health guidance
- Simple command-line interface

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Naïve Bayes Machine Learning Algorithm
